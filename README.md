# qrwolf
QR Wolf Flask Website

This is a Flask app. 

For sending emails, set up the following in the Flask config file:
  
  SECRET_KEY
  MAIL_SERVER
  MAIL_PORT 
  MAIL_USE_TLS = False
  MAIL_USE_SSL = True
  MAIL_DEBUG = True
  MAIL_USERNAME 
  MAIL_PASSWORD
  MAIL_DEFAULT_SENDER
  MAIL_MAX_EMAILS = 5
  MAIL_SUPRESS_SEND = False
  MAIL_ASCII_ATTACHMENTS = False
